XiaoMingGame
============
It should be a SLG game, maybe with some RPG elements, and it's name is XiaoMing

主要目的是实现一套JavaScript的游戏MVC框架，顺便实现一个我想了很久的游戏。

#游戏的目录结构

root + resource
     + scripts
          + app
              + commponents //系统组件
              + config //配置目录
              + controllers //控制器
              + models //模型
              + views //视图
              - main.js //游戏主入口
          + library
              + sea //seajs 加载器
              + xiaoming //小明MVC框架
              - jquery
              - kinetic
          + unit-test //测试
     + index.html
